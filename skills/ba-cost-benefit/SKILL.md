# BA Cost Benefit

Use this skill when a decision needs financial justification.

## Workflow
1. List all relevant costs, direct and indirect.
2. Identify tangible and intangible benefits.
3. Estimate timing, confidence, and assumptions.
4. Calculate ROI, payback, and break-even where useful.
5. Compare scenarios if there are alternatives.
6. State the financial recommendation plainly.

## Deliverables
- Cost-benefit summary
- ROI and payback notes
- Scenario comparison
