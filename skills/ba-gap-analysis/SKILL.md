# BA Gap Analysis

Use this skill when you need to explain what is missing, why it matters, and what should close the gap.

## Workflow
1. Capture the current state and target state.
2. List gaps by process, data, people, or technology.
3. Rate impact, effort, urgency, and dependency.
4. Find root causes and downstream effects.
5. Recommend one or more solution options.
6. Summarize risk and implementation priority.

## Deliverables
- Gap table
- Root cause notes
- Impact assessment
- Recommendation summary

- Priority reflects business impact
